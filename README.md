## What I'm Building

### 🤖 [balance_bot](https://github.com/lusher00/balance_bot)
Self-balancing inverted pendulum on BeagleBone Blue. Custom MPU-9250 DMP driver from scratch, cascaded PID (balance / position / heading), RoboClaw 2×15A motor controller, SBUS RC input, WebSocket telemetry, web dashboard, and a SwiftUI iOS companion app.

### 🔌 IIO-1000 — Industrial I/O Board
KiCad PCB design in progress. ATmega1284P, MC33978 multi-switch detect, BTS7008 high-side switches, DRV8876 motor drivers, CAN FD, isolated RS-485, W5500 Ethernet. Targeting rugged maker/industrial use with automotive-grade components.

### 📡 [TuneHUD](https://github.com/lusher00/tunehud)
Raspberry Pi 5 telemetry HUD and data visualization platform. Python asyncio WebSocket gateway with a plugin architecture bridging hardware (ODrive, Modbus, BLE, serial, CAN) to web and iOS clients.

### 👁️ [hailo-tracker](https://github.com/lusher00/hailo-tracker)
Real-time object detection on Pi 5 + Hailo-8L NPU. YOLOv8s at 30 FPS, configurable class filtering, MJPEG stream to browser. Drop-in systemd service.

### ⚙️ [beltsolve](https://lusher00.github.io/beltsolve)
Parametric timing belt/pulley solver running in the browser. Part of a growing suite of embedded engineering calculators.
